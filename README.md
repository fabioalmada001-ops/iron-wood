# IRON WOOD — PRUEBA DE SITIO WEB

Sitio estático listo para publicar con GitHub Pages.

## Contenido

- Página de inicio promocional.
- Descripción de la marca.
- Catálogo con imágenes, buscador y filtros.
- Detalle de cada producto.
- Formulario para personalizar medidas, colores y accesorios.
- Asistente de diseño de demostración.
- Propuesta lista para copiar o enviar por WhatsApp.
- Diseño claro, minimalista, rústico e industrial.
- Adaptación para computadora y celular.

## Importante sobre la inteligencia artificial

La versión actual simula un asistente dentro del navegador y genera una propuesta
a partir de los datos del formulario. No utiliza todavía una API de inteligencia
artificial real.

En una versión futura se puede conectar con una IA para:

- generar conceptos y renders;
- recomendar medidas;
- comparar variantes;
- ayudar con presupuestos;
- preparar fichas técnicas.

## Publicar en GitHub

1. Extraer el archivo ZIP.
2. Abrir el repositorio de GitHub.
3. Seleccionar `Add file` > `Upload files`.
4. Arrastrar TODO el contenido de esta carpeta:
   - index.html
   - styles.css
   - script.js
   - assets
   - README.md
5. Presionar `Commit changes`.
6. Esperar a que GitHub Pages termine el despliegue.

## Configurar WhatsApp

Buscar este número dentro de `index.html` y `script.js`:

5490000000000

Reemplazarlo por el número real con código de país, sin espacios ni símbolos.
Ejemplo Argentina:

5493764000000

## Archivos principales

- `index.html`: estructura y textos.
- `styles.css`: diseño visual.
- `script.js`: catálogo, filtros, ventanas y asistente.
- `assets/`: imágenes.
